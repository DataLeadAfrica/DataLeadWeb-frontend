UPLOAD STEPS (GitHub website, no terminal)

  1. Go to github.com/DataLeadAfrica/DataLeadWeb-frontend

  2. Make the new branch FIRST.
     Click the branch dropdown (it says "main").
     Type:  feature/kids-python-2026-page
     Click "Create branch: feature/kids-python-2026-page from main".

  3. Confirm the dropdown now shows feature/kids-python-2026-page,
     not main. Do not continue until it does.

  4. Click into the "public" folder.

  5. Click "Add file", then "Upload files".
     Do NOT use "Create new file". Pasting this much code into the
     GitHub editor corrupts it.

  6. Drag the whole kids-python-2026 FOLDER into the upload area.
     Drag the folder itself, not the file inside it. GitHub will
     create public/kids-python-2026/index.html for you.

  7. Before committing, check the file list on screen shows:
        public/kids-python-2026/index.html
     and nothing else. If READ-ME-FIRST.txt is listed, remove it.

  8. Commit message:
        Add Kids Python Bootcamp 2026 registration page
     Choose "Commit directly to the feature/kids-python-2026-page branch".
     Click "Commit changes".

  9. Wait about a minute for Vercel to build the preview.
     Go to the Vercel dashboard, open the newest deployment for this
     branch, and copy the preview URL.

 10. Open this in your browser:
        <preview-url>/kids-python-2026/
